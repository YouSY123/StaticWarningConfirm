The memory leak happens in the function "int handle(int flag)" before line 28, so we inspect line 20-27 for information:
    Buf *b = create_buf(128);
    if (!b) return -1;

    if (flag > 0) {
        b = create_buf(64);
        if (!b) return -2;
    }
"b" is allocated memory by "create_buf" function, and if "flag > 0", "b" is allocated memory twice. If pointer "b" is not freed between the two allocations and there is no alias that is used to free the memory, memory leak happens.
